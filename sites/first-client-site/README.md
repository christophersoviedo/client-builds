# First Client Site

A simple Vite + React starter project for a mobile-friendly landing page.

## Local setup

1. Change into the project directory:
   ```bash
   cd sites/first-client-site
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
